Image Detection through Machine Learning - Took a library of photos and used different algorithms such as KNN, SVM, and AdaBoost. The project looked through photos of elephant heads and chicken heads and accurately distinguished between the two with over 90% accuracy.

Local Bar Database - A PostgreSQL database that holds information regarding local bars in the area. This includes their location, demographics, and ratings, which can be modified and accessed by a query at any time.

Shuffle Cards - Using linked stacks in Java this program can generate a deck of cards, shuffle it randomly, deal, and discard, which can be adapted to any card game.
